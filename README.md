## 1. 서비스 소개
- 함께 달리는 즐거움을 나눌 수 있는 공간<br>
들숨날숨 from 멋사

- 같은 동네 혹은 서로 다른 동네에서 새로운 사람들과 함께 달리거나 소식을 공유하세요.<br>
자유롭게 날짜와 장소만 게시한다면 누구든 함께 달릴 수 있습니다!<br>
내가 달린 기록과 사람들을 이웃들에게 자유롭게 공유할 수 있습니다!
